# Standard Library Documentation

This directory contains documentation for the KODEON standard library.

## Directory Structure

-   [api/](api/) - API reference for standard library modules
-   [guides/](guides/) - Usage guides for standard library modules
-   [tutorials/](tutorials/) - Tutorials for standard library modules

## Documentation Categories

### API Reference ([api/](api/))

Detailed API documentation for each standard library module, including:

-   Function signatures
-   Parameter descriptions
-   Return value specifications
-   Error conditions
-   Usage examples

### Guides ([guides/](guides/))

Usage guides for standard library modules, including:

-   Module overview
-   Common use cases
-   Best practices
-   Performance considerations

### Tutorials ([tutorials/](tutorials/))

Step-by-step tutorials for learning standard library modules, including:

-   Basic usage
-   Intermediate concepts
-   Advanced features
-   Integration examples

## Module Categories

The standard library documentation is organized by module category:

1. **Core Modules** - Essential functions (types, math)
2. **Data Modules** - Data structures and formats (collections, JSON)
3. **System Modules** - System interaction (IO, time, concurrency, file system)
4. **Web Modules** - Web development and networking
5. **Security Modules** - Security and cryptography
6. **Utility Modules** - Utility functions (logging)

## Multi-language Support

Documentation is available in both Indonesian and English:

-   Files in Indonesian use the `.id.md` extension
-   Files in English use the `.en.md` extension
-   Some files contain both languages for comparison
